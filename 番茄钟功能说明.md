## 🐛 常见问题

### Q1: 显示"网络错误，请检查后端"
**原因**: 后端服务没有运行  
**解决**: 启动后端服务（见步骤1）

### Q2: 显示"记录失败，请重试"
**原因**: 数据库表可能不存在  
**解决**: 
```bash
cd "d:\tingting\xixi\计算机设计大赛\ZhiNote2.0\ZhiNote2.0hou"
python -X utf8 create_users_table.py
```

### Q3: 记录成功但学习报告没有变化
**原因**: 学习报告页面没有刷新  
**解决**: 下拉刷新或重新进入学习报告页面

### Q4: Console显示"时长不足1分钟，不记录"
**原因**: 番茄钟没有运行就重置了  
**解决**: 让番茄钟正常运行至少1分钟
